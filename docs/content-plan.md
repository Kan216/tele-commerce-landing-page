- **Eyebrow**: `Telegram Shop Tool`

- **Sub-heading**: Customer တွေအတွက် — tap တစ်ချက်နဲ့ order တင်နိုင်ပါတယ်။ **Seller တွေအတွက် — screenshot ရှာစရာ၊ order ရေးမှတ်စရာ မလိုတော့ပါ။**

- **CTA Micro-text**: `Setup — 5 minutes / Tech knowledge မလိုပါ`
- **Receipt Hero Graphic**: Order `#TC-0842` ticket with `CONFIRMED` stamp.

### How It Works Section
- **Label**: `How it works`
- **Heading**: Setup — ၃ ဆင့်ပဲ ရှိပါတယ်
- **User Guide Link**: [Setup User Guide အပြည့်အစုံ ကြည့်ရန် →](https://drive.google.com/file/d/1mLACAjHxuTWQZg4ngQ3yOTwhZdzfUfui/view?usp=sharing)
- **Steps**:
  1. `01` — **Product ထည့်ပါ**: ဈေးနှုန်း၊ ပုံ၊ description နဲ့ product list တစ်ခု bot ထဲမှာ ချက်ချင်း ပြင်လို့ရပါတယ်။
  2. `02` — **Chat Bot ရယူပါ**: Link တစ်ခုတည်းကို Telegram group description၊ bio စတဲ့ နေရာတိုင်းမှာ ချိတ်ထားလို့ရပါတယ်။
  3. `03` — **Customer တို့ အော်တို order တင်**: Customer က link ကို tap ပြီး order တင်တာနဲ့ seller ဆီ clean order summary ချက်ချင်း ရောက်ပါတယ်။

## Tab 2: Features (`#features`)

- **Section Label**: `Capabilities`
- **Section Heading**: Telegram Shop အတွက် လိုအပ်သမျှ အကုန်ပါဝင်ပါတယ်
- **Sub-heading**: Website ဖွင့်စရာမလိုဘဲ Telegram Bot + Dashboard တစ်ခုတည်းနဲ့ အရောင်းစနစ်တစ်ခုလုံးကို စီမံလိုက်ပါ။

### Feature Cards (Scannable Cards with Burmese Headlines & Plain Actionable Descriptions)

1. 📦 **Product Catalog Management (ကုန်ပစ္စည်း စီမံခန့်ခွဲမှု)**
   - **Burmese Headline**: Product များကို ပုံ၊ ဈေးနှုန်း၊ Stock နဲ့တကွ လွယ်ကူစွာ စီမံပါ
   - **Description**: Product အသစ်များ ထည့်သွင်းခြင်း၊ ဓာတ်ပုံ တင်ခြင်း၊ ဈေးနှုန်းနှင့် Stock အရေအတွက်များကို Dashboard မှ ချက်ချင်း အချိန်မရွေး ပြင်ဆင်နိုင်သည်။

2. 🤖 **Telegram Shop Bot (အလိုအလျောက် ရောင်းချပေးမည့် Bot)**
   - **Burmese Headline**: Customer များ Telegram Bot အတွင်းမှ တိုက်ရိုက် ကြည့်ရှု အော်ဒါတင်နိုင်
   - **Description**: Customer များသည် Telegram Bot ထဲတွင် Product အသေးစိတ်ကြည့်နိုင်ပြီး Tap တစ်ချက်တည်းဖြင့် တိုက်ရိုက် Order တင်နိုင်သည်။

3. 🛒 **Automatic Order Numbering & Status (Order စနစ်ကျ စီမံမှု)**
   - **Burmese Headline**: Order တိုင်းအတွက် Auto Number (#TC-xxxx) နှင့် Status Tracking ပါဝင်
   - **Description**: Order သစ်များကို Dashboard တွင် ချက်ချင်းမြင်ရပြီး Pending၊ Confirmed၊ Completed စသည့် Status များကို စနစ်တကျ ပြောင်းလဲ စီမံနိုင်သည်။

4. 💳 **Payment Slip Verification (ငွေလွှဲ ပြေစာ စစ်ဆေးမှု)**
   - **Burmese Headline**: KPay / Wave / Bank Receipt Screenshot များကို Dashboard တွင် အတည်ပြုပါ
   - **Description**: Customer မှ ငွေလွှဲ ပြေစာ တင်လိုက်သည်နှင့် Shop Owner က Dashboard တွင် Receipt ကို အမှားအယွင်းမရှိ စစ်ဆေးပြီး အတည်ပြုနိုင်သည်။

5. 👥 **Customer History & Database (ဝယ်ယူသူ မှတ်တမ်း စုဆောင်းမှု)**
   - **Burmese Headline**: Customer တစ်ဦးချင်းစီ၏ Order History ကို စုစည်းထားရှိနိုင်
   - **Description**: Telegram User အကောင့်များ၏ ဝယ်ယူမှု မှတ်တမ်းနှင့် အချက်အလက်များကို Dashboard တွင် တစ်နေရာတည်း စုစည်း သိမ်းဆည်းပေးထားသည်။

6. 📊 **Unified Shop Dashboard (ဗဟိုချုပ်ကိုင်မှု Dashboard)**
   - **Burmese Headline**: Product၊ Order၊ Customer နှင့် Sales Summary များကို တစ်နေရာတည်း မြင်ကွင်း
   - **Description**: Shop အရောင်း အခြေအနေများကို Dashboard တွင် အချိန်နှင့်တပြေးညီ ကြည့်ရှုနိုင်ပြီး အချိန်ကုန်သက်သာစွာ လုပ်ငန်းကို လည်ပတ်နိုင်သည်။

## Tab 4: FAQ & Contact (`#faq-contact`)

- **Section Label**: `Support & Contact`
- **Section Heading**: မကြာခဏ မေးလေ့ရှိသော မေးခွန်းများနှင့် ဆက်သွယ်ရန်

### FAQ Items (Official 5 Questions from Tele-Commerce Bot & Dashboard)

1. **Q: Tele-Commerce ဆိုတာ ဘာလဲ?**
   - **A**: Tele-Commerce က Telegram အတွင်းမှာပဲ သင့်ဆိုင်ရဲ့ Product တွေကို ပြသနိုင်၊ Order လက်ခံနိုင်၊ Customer တွေနဲ့ Chat လုပ်နိုင်တဲ့ Commerce Platform ဖြစ်ပါတယ်။ App ထပ်သုံးစရာမလိုဘဲ Telegram ကနေ အကုန်စီမံနိုင်ပါတယ်။

2. **Q: Website မရှိလည်း အသုံးပြုလို့ရလား?**
   - **A**: ရပါတယ်။ Website မရှိလည်း Telegram Bot တစ်ခုနဲ့ Product Catalog၊ Order Management နဲ့ Customer Management ကို Tele-Commerce ကနေ တိုက်ရိုက်အသုံးပြုနိုင်ပါတယ်။

3. **Q: Customer က Order ဘယ်လိုတင်ရမလဲ?**
   - **A**: Customer က Telegram Bot ထဲဝင်ပြီး Product ကြည့်နိုင်ပါတယ်။ Order တင်ပြီး ငွေလွှဲပြီး Receipt ကို Bot ထဲမှာပဲ ပို့လိုက်ရုံနဲ့ Shop Owner ဘက်က Dashboard မှာ Order ကို စစ်ဆေးပြီး အတည်ပြုနိုင်ပါတယ်။

4. **Q: Product ဘယ်နှစ်ခုတင်လို့ရလဲ?**
   - **A**: Product အများဆုံး (100) ခုထိ ထည့်သွင်းနိုင်ပြီး Product ပုံတစ်ပုံစီအတွက် File Size Limit လည်း သတ်မှတ်ထားပါတယ်။

5. **Q: Tele-Commerce က ဘယ်လိုစီးပွားရေးမျိုးတွေအတွက် သင့်တော်လဲ?**
   - **A**: Online Shop၊ Fashion၊ Cosmetic၊ Food & Beverage၊ Electronics၊ Digital Products နဲ့ Telegram ကနေ Order လက်ခံချင်တဲ့ Business အများစုအတွက် သင့်တော်ပါတယ်။ Coding မသိလည်း အလွယ်တကူ စတင်အသုံးပြုနိုင်ပါတယ်။
